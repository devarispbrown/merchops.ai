# Deployment Quick Start

5-minute guide to deploy MerchOps to production.

## Fastest Path: Render (15 minutes)

### Prerequisites

- [ ] GitHub account
- [ ] Shopify Partner account + app created
- [ ] Anthropic or OpenAI API key
- [ ] Resend or SendGrid API key

### Steps

1. **Push to GitHub**
   ```bash
   git remote add origin https://github.com/your-org/merchops.git
   git push -u origin main
   ```

2. **Deploy to Render**
   - Go to [render.com](https://render.com)
   - Sign up/login with GitHub
   - Click "New" → "Blueprint"
   - Select your repository
   - Render detects `render.yaml` automatically
   - Click "Deploy"

3. **Set Environment Variables**

   In Render Dashboard, set these for **both** `merchops-web` and `merchops-worker`:

   ```bash
   # Your app URL (provided by Render)
   NEXTAUTH_URL=https://merchops-web.onrender.com

   # Shopify OAuth (from partners.shopify.com)
   SHOPIFY_CLIENT_ID=your_shopify_client_id
   SHOPIFY_CLIENT_SECRET=your_shopify_client_secret

   # AI Provider
   ANTHROPIC_API_KEY=sk-ant-api03-...
   # OR
   OPENAI_API_KEY=sk-proj-...

   # Email Provider
   EMAIL_PROVIDER_API_KEY=re_...

   # Optional: Error Tracking
   SENTRY_DSN=https://...@sentry.io/...
   ```

   **Note:** `NEXTAUTH_SECRET`, `ENCRYPTION_KEY`, and `SHOPIFY_WEBHOOK_SECRET` are auto-generated by Render.

4. **Run Migrations**

   After services deploy:
   - Go to `merchops-web` service
   - Click "Shell" tab
   - Run:
     ```bash
     pnpm prisma migrate deploy
     ```

5. **Configure Shopify App**

   In [Shopify Partners Dashboard](https://partners.shopify.com):
   - Apps → Your App → App Setup
   - Set "App URL": `https://merchops-web.onrender.com`
   - Add "Allowed redirection URL":
     ```
     https://merchops-web.onrender.com/api/auth/callback/shopify
     ```
   - Save

6. **Verify Deployment**

   ```bash
   # Health check
   curl https://merchops-web.onrender.com/api/health
   # Expected: {"status":"ok","timestamp":"..."}

   # Visit app
   open https://merchops-web.onrender.com
   ```

7. **Test Flow**
   - Sign up for account
   - Connect Shopify store
   - Wait for initial sync (1-5 minutes)
   - Verify opportunities appear

**Done!** Your MerchOps instance is live.

## Alternative: Railway (20 minutes)

### Steps

1. **Install Railway CLI**
   ```bash
   npm install -g @railway/cli
   railway login
   ```

2. **Create Project**
   ```bash
   railway init
   ```

3. **Add Services**
   ```bash
   railway add postgresql
   railway add redis
   ```

4. **Set Environment Variables**
   ```bash
   railway variables set NEXTAUTH_SECRET=$(openssl rand -base64 32)
   railway variables set NEXTAUTH_URL=https://your-app.railway.app
   railway variables set SHOPIFY_CLIENT_ID=your_id
   railway variables set SHOPIFY_CLIENT_SECRET=your_secret
   railway variables set ENCRYPTION_KEY=$(openssl rand -hex 32)
   railway variables set ANTHROPIC_API_KEY=your_key
   railway variables set EMAIL_PROVIDER_API_KEY=your_key
   railway variables set SHOPIFY_WEBHOOK_SECRET=$(openssl rand -base64 32)
   ```

5. **Deploy**
   ```bash
   railway up
   ```

6. **Add Worker Service**
   - Railway Dashboard → "New Service"
   - From Existing Repo → your repo
   - Custom start command: `pnpm workers`
   - Share environment variables with web

7. **Run Migrations**
   ```bash
   railway run pnpm prisma migrate deploy
   ```

8. **Configure Shopify** (same as Render step 5)

9. **Verify** (same as Render step 6)

## Alternative: Fly.io (30 minutes)

### Steps

1. **Install Fly CLI**
   ```bash
   curl -L https://fly.io/install.sh | sh
   flyctl auth login
   ```

2. **Launch App**
   ```bash
   flyctl launch
   # Select region, PostgreSQL, Redis (via Upstash)
   ```

3. **Set Secrets**
   ```bash
   flyctl secrets set \
     NEXTAUTH_SECRET=$(openssl rand -base64 32) \
     NEXTAUTH_URL=https://merchops.fly.dev \
     SHOPIFY_CLIENT_ID=your_id \
     SHOPIFY_CLIENT_SECRET=your_secret \
     ENCRYPTION_KEY=$(openssl rand -hex 32) \
     ANTHROPIC_API_KEY=your_key \
     EMAIL_PROVIDER_API_KEY=your_key \
     SHOPIFY_WEBHOOK_SECRET=$(openssl rand -base64 32)
   ```

4. **Deploy**
   ```bash
   flyctl deploy
   ```

5. **Deploy Worker**
   ```bash
   flyctl apps create merchops-worker
   flyctl deploy --config fly-worker.toml --command "pnpm workers"
   ```

6. **Run Migrations**
   ```bash
   flyctl ssh console
   pnpm prisma migrate deploy
   exit
   ```

7. **Configure Shopify** (same as Render step 5)

8. **Verify** (same as Render step 6)

## Troubleshooting

### Build fails with "Cannot find module '@prisma/client'"

**Solution:** Ensure build command includes `pnpm prisma:generate`:
```bash
pnpm install && pnpm prisma:generate && pnpm build
```

### Health check returns 500

**Solutions:**
1. Check `DATABASE_URL` is set correctly
2. Verify database is running
3. Check application logs for errors

### OAuth flow fails

**Solutions:**
1. Verify `NEXTAUTH_URL` matches actual domain
2. Check Shopify redirect URLs are correct
3. Ensure `SHOPIFY_CLIENT_ID` and `SHOPIFY_CLIENT_SECRET` match Shopify

### Webhooks not arriving

**Solutions:**
1. Verify webhooks registered in Shopify Admin
2. Check webhook URL matches `NEXTAUTH_URL/api/webhooks/shopify`
3. Test webhook delivery in Shopify Admin
